# 📚 Projeto Final React (grupo 6)

Projeto desenvolvido por **Alexandre Licio, João Pedro Breves, Lucca Zapalla e Enzo Converso** utilizando **React + Vite**, **Axios**, **Context API** e **Bootstrap**, integrando autenticação, busca de livros, carrinho de compras e design responsivo.

## 🚀 Como rodar o projeto
```bash
git clone <https://github.com/Xande-05/Serratec-React-TrabalhoFinal.git>
cd ProjetoFinalReact
npm install
npm install bootstrap react-bootstrap
npm run dev

⚠️ É necessário ter Node.js e npm instalados.

```

# 🧩 Estrutura e funções

Enzo – Autenticação e estrutura geral (rotas, login, register, contexts)

Lucca – Catálogo e busca de livros (Google Books API)

João – Carrinho de compras (CRUD com MockAPI)

Alexandre – Design, componentes e modo escuro (UI/UX)

# 🛠️ Tecnologias

React, Vite, React Router DOM, Axios, Context API, Bootstrap, React-Bootstrap, Google Books API, MockAPI.
